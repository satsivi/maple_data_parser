🍁 Event Stat Optimizer
How this works
  1. Load FD data first: on the Stat Parser tab, click the copy button under FD in Scouter's Stat Efficiency section, paste it in, and parse.
  2. Scouter's export doesn't include Buff Duration or Crit Rate. Grab those FD values yourself and type them into the matching line's FD weight field.
  3. To force a skill to a specific level, set Reserved Levels for that line. Add a Complete by wk deadline if it needs to hit that level by a certain week.
  4. Already partway through the event? Fill in Current level for whatever you've already bought. The tool estimates what week you're on from those points and optimizes only what's left.
